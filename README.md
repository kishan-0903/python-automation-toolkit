# Python Automation & Logic Toolkit 🐍

This repository contains a suite of production-ready Python scripts designed to automate financial calculations, data validation, and logic-based reporting. 

## 🛠️ Core Capabilities
* **Automated Financial Reporting:** Scripts for calculating ROI and multi-video revenue streams.
* **Data Validation:** Logic-based error handling for user inputs and expense tracking.
* **Efficiency Tools:** Currency conversion and bulk data processing using Python lists and loops.

## 📁 Project Highlights

### 01. Revenue Forecast Engine (`01_youtube_calculator.py`)
A tool designed for digital creators to forecast earnings based on view-count metrics and goal-oriented logic.

### 02. Bulk Data Analyzer (`02_bulk_view_analyzer.py`)
Processes multiple data points simultaneously to provide a percentage-based progress report against financial targets.

### 03. Secure Expense Tracker (`03_smart_expense_tracker.py`)
A financial management script featuring robust error-handling to prevent negative data entry and automate budget-vs-actual analysis.

### 04. International Payment Processor (`04_currency_converter.py`)
An automated currency conversion tool using functional programming. It processes multiple foreign currency entries and converts them to local currency (INR) using standardized exchange rate logic.

## 🚀 Technical Stack
* **Language:** Python 3.x
* **Concepts:** Data Structures (Lists), Control Flow (If/Else, Loops), Functional Programming.
